# Placeholding component

It is a component that return a string of `HTML tags` that occupies a position for another components to be rendered later.

```js
const Placeholder = ({id, className}) => {
  return `
    <div 
      id="${id}"
      class="${className}" 
    ></div>`;
}

//usage
<Placeholder id="page" />
<Placeholder id="modal" className="hidden" />
<Placeholder id="notification" className="hidden" />
```
